## Project setup

- Install ruby 2.7
  - Arch/Manjaro: Follow this [Arch Wiki](https://wiki.archlinux.org/title/RVM)

- Install bundler
  - `gem install bundler:2.1.4` 
- Install required gems
  - `bundler install`
- Run server
  - `bundle exec jekyll serve`
